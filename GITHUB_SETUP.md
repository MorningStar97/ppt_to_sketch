# GitHub 仓库设置指南

## 📝 仓库描述
```
🎨 一个基于Django的PPT到Sketch转换服务，支持高保真度转换，包含Web界面和RESTful API
```

## 🏷️ 推荐标签 (Topics)
在GitHub仓库页面的About部分添加以下标签：

### 核心功能标签
- `ppt-to-sketch`
- `powerpoint-converter`
- `sketch-generator`
- `file-converter`
- `design-tools`

### 技术栈标签
- `django`
- `python`
- `django-rest-framework`
- `python-pptx`
- `pillow`
- `bootstrap`
- `javascript`

### 用途标签
- `conversion-tool`
- `design-automation`
- `web-service`
- `api-service`
- `async-processing`

### 语言标签
- `chinese`
- `中文文档`

## 🌐 网站链接
如果部署了在线演示，可以在About部分添加：
- 演示网站链接
- 文档链接

## 📋 设置步骤

1. 进入GitHub仓库页面
2. 点击右侧的"About"齿轮图标
3. 添加描述信息
4. 在"Topics"中添加上述标签
5. 保存设置
